## Style Guide

### **Layout**

The designs were created to the following widths:

> - Mobile: 375px
> - Desktop: 1440px

### **Colors**

> - Background: #f9f3ef
> - Text color: #4a4a4a
> - Text color light: #666
> - Heading color: #333
> - Pink: #7B284FFF
> - Pink light: #fbeff3
> - Brown: #854632FF
> - Orange: #b45f06
> - Border color: #ddd

> - Font size (paragraph): 16px

> - Font size (Heading 3): 18px
